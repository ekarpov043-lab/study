# Медицинский VK Чат-бот «Клиника в смартфоне»

Чат-бот для ВКонтакте. Позволяет пациентам отслеживать здоровье,
участвовать в челленджах, получать достижения и скидки.

---

## Быстрый старт

```bash
# 1. Клонировать
git clone <url>
cd medical_bot

# 2. Установить зависимости
pip install -r requirements.txt

# 3. Настроить окружение
cp .env.example .env
# Отредактировать .env — вставить VK_GROUP_TOKEN и параметры БД

# 4. Создать базу данных
psql -U postgres -c "CREATE DATABASE medical_bot;"

# 5. Заполнить начальными данными
python seed_db.py

# 6. Запустить бота
python bot.py
```

## Переменные окружения (.env)

| Переменная | Описание | По умолчанию |
|-----------|----------|-------------|
| `VK_GROUP_TOKEN` | Токен группы ВК (обязательно) | — |
| `DB_HOST` | Хост PostgreSQL | localhost |
| `DB_PORT` | Порт PostgreSQL | 5432 |
| `DB_NAME` | Имя базы данных | medical_bot |
| `DB_USER` | Пользователь БД | postgres |
| `DB_PASSWORD` | Пароль БД | postgres |
| `LOG_FILE` | Файл логов | logs/bot.log |
| `LOG_LEVEL` | Уровень логирования | INFO |

## Логирование

Логи пишутся в `logs/bot.log` с ротацией:
- Максимальный размер файла: 10 MB
- Количество файлов: 5
- Формат: `[2024-01-15 10:23:45] [INFO] [handlers.health_log] msg`

Уровни:
- `INFO` — действия пользователей, начисления баллов
- `WARNING` — некорректный ввод
- `ERROR` — ошибки БД и VK API
